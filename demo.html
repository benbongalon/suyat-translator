isawika.com

Demo app for showing how to use the suyat.js translator library

Select a target script, then type some Tagalog/Mangyan/etc text and watch the live translation. Notice how the suyat symbols dance. Kudlit (diacritical) marks will flicker above or below the symbols like fireflies, and the symbols themselves will sometimes coalesce or divide.

Try typing "nga" or "mga" and see how the symbols shape-shift.

Suyat Translator

If you see [][][] instead of suyat symbols, add the missing fonts to your browser:

Windows
  1. Download the Suyat fonts package (.zip)
  2. Uncompress the package
  3. Search for “fonts” in the Start Menu or go to Start → Control Panel → Appearance and Personalization → Fonts
  4. Select all fonts and drag them into the Fonts folder
Mac OS
  1. Download the Suyat fonts package (.zip)
  2. Double click on the package to uncompress it
  3. Open Font Book (Go to Finder → Applications → Font Book)
  4. Select all of the font files and drag them to the Font column (i.e., the second column) of Font Book
Linux
  1. Download the Suyat fonts package (.zip)
  2. Follow the rest of the Linux instructions in Noto Fonts site

Known Issues

  • The app gets confused if you add/delete characters in the middle of the text. As a workaround, delete from the end until you reach the edit point then append from there. Or refresh the page to start over.
